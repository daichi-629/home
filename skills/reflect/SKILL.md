# Reflect — セッション振り返りの恒久化

1. このセッションでユーザーから受けた訂正・指摘・確定した方針・環境固有の詰まりを列挙する。
2. それぞれ書き込み先を選ぶ:
   - vault (~/Documents/vault) 内での作業 → vault の ai/MEMORY.md。参照したノート自体に問題があった場合はそのノートも修正する。
   - プロジェクト固有の設計原則・規約 → そのリポジトリの CLAUDE.md / AGENTS.md
   - プロジェクト横断の環境固有知見 → ~/.agent-memo/(1事実1ファイル + MEMORY.md index更新)
3. 既存エントリと重複しないか確認し、重複時は新規作成せず更新する。相対日付は絶対日付に変換する。
4. 書き込んだファイルと内容の一覧をユーザーに報告する。
